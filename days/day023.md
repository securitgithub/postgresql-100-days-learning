# Day 023: 字符串函数

## 学习目标

- 掌握 PostgreSQL 常用字符串度量与转换函数：length、upper、lower、trim
- 熟练使用字符串截取与查找函数：substring、position、replace
- 理解字符串拼接方式：concat、concat_ws 以及 `||` 运算符的区别
- 掌握字符串拆分与聚合函数：split_part、string_agg
- 了解 PostgreSQL 正则表达式函数的基本用法

## 理论讲解

### 字符串度量函数

| 函数 | 说明 | 示例 |
|------|------|------|
| `length(str)` | 返回字符数（非字节数） | `length('你好') → 2` |
| `char_length(str)` | 同 length，SQL 标准写法 | `char_length('abc') → 3` |
| `octet_length(str)` | 返回字节数 | `octet_length('你好') → 6`（UTF-8） |

### 大小写转换

- `upper(str)` — 转为大写
- `lower(str)` — 转为小写
- `initcap(str)` — 每个单词首字母大写

### TRIM 函数族

```sql
trim(str)              -- 去除两端空白
ltrim(str)             -- 去除左侧空白
rtrim(str)             -- 去除右侧空白
trim(leading 'x' from str)   -- 去除左侧指定字符
trim(both 'xy' from str)     -- 去除两端指定字符
```

### 截取与查找

- `substring(str FROM start FOR length)` — 截取子串（位置从 1 开始）
- `left(str, n)` / `right(str, n)` — 从左 / 右取 n 个字符
- `position(sub IN str)` — 返回子串首次出现的位置，未找到返回 0
- `strpos(str, sub)` — 同 position，参数顺序不同

### 替换与填充

- `replace(str, from, to)` — 替换所有匹配的子串
- `overlay(str PLACING new FROM start FOR count)` — 覆盖替换
- `lpad(str, len, fill)` / `rpad(str, len, fill)` — 左 / 右填充到指定长度
- `repeat(str, n)` — 重复字符串 n 次
- `reverse(str)` — 反转字符串

### 拼接函数

- `||` 运算符：如果任一操作数为 NULL，结果为 NULL
- `concat(val1, val2, ...)` ：忽略 NULL 值进行拼接
- `concat_ws(sep, val1, val2, ...)` ：使用分隔符拼接，忽略 NULL

### 拆分与聚合

- `split_part(str, delimiter, n)` — 按分隔符拆分后取第 n 部分（从 1 开始）
- `string_to_array(str, delimiter)` — 拆分为数组
- `string_agg(expression, delimiter)` — 聚合函数，将多行值拼接为一个字符串

### 正则表达式函数

- `regexp_match(str, pattern)` — 返回第一个匹配（数组）
- `regexp_matches(str, pattern, 'g')` — 返回所有匹配（集合）
- `regexp_replace(str, pattern, replacement)` — 正则替换
- `regexp_split_to_table(str, pattern)` — 正则拆分为行
- `regexp_split_to_array(str, pattern)` — 正则拆分为数组

### 示例一：度量与大小写

```sql
SELECT
    name,
    length(name) AS name_len,
    upper(name) AS upper_name,
    lower(name) AS lower_name,
    initcap(name) AS initcap_name
FROM contacts;
```

### 示例二：TRIM 清理空白

```sql
-- 去除前后空格
SELECT
    name,
    trim(name) AS trimmed,
    length(name) AS original_len,
    length(trim(name)) AS trimmed_len
FROM contacts
WHERE name LIKE ' %' OR name LIKE '% ';
```

### 示例三：截取与查找

```sql
-- 提取邮箱的用户名和域名部分
SELECT
    email,
    -- 方法一：使用 position + substring
    substring(email FROM 1 FOR position('@' IN email) - 1) AS username,
    substring(email FROM position('@' IN email) + 1) AS domain,
    -- 方法二：使用 split_part
    split_part(email, '@', 1) AS username_v2,
    split_part(email, '@', 2) AS domain_v2
FROM contacts
WHERE email IS NOT NULL;
```

### 示例四：替换与格式化

```sql
-- 将电话号码中的连字符去除
SELECT
    phone,
    replace(phone, '-', '') AS clean_phone,
    -- 手机号脱敏：保留前3后4，中间用****替代
    overlay(replace(phone, '-', '') PLACING '****' FROM 4 FOR 4) AS masked_phone
FROM contacts;
```

### 示例五：拼接函数对比

```sql
-- || 运算符：遇到 NULL 结果为 NULL
SELECT name || ' <' || email || '>' AS contact_info_v1
FROM contacts;

-- concat 函数：忽略 NULL
SELECT concat(name, ' <', email, '>') AS contact_info_v2
FROM contacts;

-- concat_ws 函数：使用分隔符拼接，忽略 NULL
SELECT concat_ws(' | ', trim(name), email, phone) AS contact_info_v3
FROM contacts;
```

### 示例六：LEFT / RIGHT / LPAD / RPAD

```sql
SELECT
    trim(name) AS name,
    left(trim(name), 5) AS first_5,
    right(trim(name), 3) AS last_3,
    lpad(CAST(id AS TEXT), 6, '0') AS padded_id,  -- 000001, 000002 ...
    rpad(trim(name), 20, '.') AS name_padded       -- 右侧用点填充到20字符
FROM contacts;
```

### 示例七：string_agg 聚合

```sql
-- 将每个域名下的所有用户聚合为逗号分隔的字符串
SELECT
    split_part(email, '@', 2) AS domain,
    string_agg(trim(name), ', ' ORDER BY name) AS users
FROM contacts
WHERE email IS NOT NULL
GROUP BY split_part(email, '@', 2);
```

### 示例八：正则表达式

```sql
-- 提取地址中的城市名（以"市"结尾的部分）
SELECT
    address,
    (regexp_match(address, '(\S+市)'))[1] AS city
FROM contacts
WHERE address IS NOT NULL;

-- 正则替换：去除所有非数字字符
SELECT
    phone,
    regexp_replace(phone, '[^0-9]', '', 'g') AS digits_only
FROM contacts;

-- 正则拆分地址为多行
SELECT
    trim(name) AS name,
    regexp_split_to_table(address, '\s+') AS address_part
FROM contacts
WHERE address IS NOT NULL;
```

### 示例九：repeat 和 reverse

```sql
SELECT
    repeat('*', score / 10) AS bar_chart,
    reverse('PostgreSQL') AS reversed
FROM (VALUES (92), (78), (55)) AS t(score);
```

## 常见问题

### Q1：length() 返回的是字符数还是字节数？
**A**：`length()` 返回的是字符数。在 UTF-8 编码下，一个中文字符算 1 个字符但占 3 个字节。如果需要字节数，使用 `octet_length()`。

### Q2：`||` 拼接和 concat 函数哪个更好？
**A**：如果确定所有操作数不为 NULL，`||` 更简洁。但如果可能存在 NULL 值，推荐使用 `concat` 或 `concat_ws`，它们会自动忽略 NULL 而非返回 NULL。

### Q3：正则表达式的 'g' 标志是什么意思？
**A**：`'g'` 表示全局匹配（global），即匹配所有出现的位置而非仅第一个。`regexp_replace` 不加 `'g'` 只替换第一次匹配，`regexp_matches` 不加 `'g'` 只返回第一个匹配。
